# 2026-06-10 — Day 5 Hello API

## What I Learned

- Cách chuyển một FastAPI hello world một file thành service có cấu trúc rõ hơn.
- Cách dùng `create_app()` factory để app dễ test và dễ mở rộng.
- Cách tách route theo module trong `app/routes/`.
- Cách dùng Pydantic schema để validate request body và định nghĩa response model.
- Cách dùng `pydantic-settings` để đọc config từ env vars như `APP_VERSION`, `PORT`, `LOG_LEVEL`.
- Cách viết middleware log mỗi request với `method`, `path`, `status`, `duration_ms`.
- Cách viết test cho happy path và edge case `422`.
