## What I Learned

- Day 7 tập trung vào PostgreSQL thuần trước khi dùng SQLAlchemy.
- Primary key định danh row, foreign key mô tả quan hệ giữa bảng.

## Day 8 Update

### What I Learned

- SQLAlchemy 2.0 dùng typed style: `Mapped[...]` cho type ORM và `mapped_column(...)` cho column trong DB.
- `AsyncSession` nên được tạo theo từng request qua FastAPI dependency.
- `flush()` gửi SQL xuống DB để lấy id nhưng chưa commit transaction.
- `relationship()` giúp biểu diễn quan hệ Python giữa models như `User.posts` và `Post.author`.
- N+1 query xảy ra khi lazy-load relationship từng row; `selectinload()` preload relationship bằng query phụ.
